This is a Zurich Assessment web run on Next JS, Redux and authenticated tool Google OAuth2

## Getting Started

1. run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

2. Open (http://localhost:3000/login) with your browser to see the start of login page.

## Module Involve
- Login
- User Listing
